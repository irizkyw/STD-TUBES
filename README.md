# Tugas Besar Struktur Data

Tugas ini adalah Tugas akhir matakuliah Struktur data dengan materi Multi Linked List(M to N) dengan tema Cargo.

## Usage

- Buka ```TUGAS_BESAR.sln``` dengan Visual Studio 2019/2022
